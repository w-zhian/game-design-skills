# 三国群英传玩法框架设计

> 来源：sanguo-troop-framework skill
> 用途：一队三将/统率技/兵种克制/异步PVP的玩法设计框架
> 版本：v3.0

## 核心机制：统帅值系统

```
总消耗 = 武将统帅值 + 士兵数 × 每兵消耗（默认4点）
统帅值上限：100
流派效果 = 基础值 × (1 + 统帅值/1000)
```

## 武将三流派设计

| 流派 | 效果 | 代表武将 |
|------|------|---------|
| 攻击系 | 攻击+%，暴击率+%，暴击伤害+% | 关羽/赵云/马超/黄忠/魏延 |
| 防御系 | 防御+%，生命+%，减伤+% | 吕布/张飞/典韦/许褚/太史慈 |
| 战术系 | 技能伤害+%，冷却缩减+%，能量恢复+% | 曹操/刘备/孙权/司马懿/诸葛亮 |

**核心原则**：任何武将可携带任何兵种（剑/枪/刀任选，完全解耦）

## 武将协同效果矩阵（1+1+1>3）

### 双武将组合
| 组合 | 效果 |
|------|------|
| 攻击+防御 | 战意联动：攻击系暴击→防御系追击+30% |
| 攻击+战术 | 穿透战意：攻击系技能→战术系伤害+40%，冷却-20% |
| 防御+战术 | 坚守阵地：防御系承伤→战术系蓄力→技能+15%/层 |

### 同流双武将
| 组合 | 效果 |
|------|------|
| 双攻击 | 狂暴战意：暴击→追击+15%/层（5层），正反馈循环 |
| 双防御 | 不动如山：承伤→防御+8%/层（8层）+大伤害减半50% |
| 双战术 | 连环计：技能→冷却-15%/次（3层）+范围+10%/层 |

### 三武将组合
| 组合 | 效果 |
|------|------|
| 攻击+防御+战术 | 三国之魂：激活所有双武将效果+每30秒全场士兵攻+15%/防+10%持续5秒 |

**实战价值**：三流齐全→双武将效果全激活+周期爆发BUFF→总输出+40~60%，总防御+20~30%

## 兵种差异化

| 兵种 | 攻击权重 | 防御权重 | 血量权重 | 定位 |
|------|---------|---------|---------|------|
| 剑兵 | 0.9 | 1.3 | 1.1 | 防御特化 |
| 枪兵 | 1.3 | 0.8 | 0.9 | 攻击特化 |
| 刀兵 | 1.0 | 1.0 | 1.3 | 血量特化 |

**兵种克制**：枪克剑(+30%)、剑克刀(+30%)、刀克枪(+30%)

## 设计启示

1. **流派与兵种解耦**：武将流派是战斗定位，兵种是属性倾向，两者独立选择
2. **协同是核心驱动**：玩家组建队伍的动力不是"我要最强的武将"，而是"我要最强的组合"
3. **三流齐全是最优解但不是唯一解**：双同流也有独特效果，保证Build多样性
